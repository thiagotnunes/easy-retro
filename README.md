# Easy Retro

Virtual Card Wall aimed for agile retrospectives

{<img src="http://travis-ci.org/thiagotnunes/easy-retro.png" />}[http://travis-ci.org/thiagotnunes/easy-retro.png]

## Installing

    $ gem install bundler
    $ bundle

## Running tests

    $ bundle exec rake spec

## Running application

    $ ruby lib/easy_retro.rb
