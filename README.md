# Easy Retro

Virtual Card Wall aimed for agile retrospectives

[![](http://travis-ci.org/thiagotnunes/easy-retro.png)](http://travis-ci.org/thiagotnunes/easy-retro)

## Installing

    $ gem install bundler
    $ bundle

## Running tests

    $ bundle exec rake spec

## Running application

    $ ruby lib/easy_retro.rb
