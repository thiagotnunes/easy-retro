# Easy Retro

Virtual Card Wall aimed for agile retrospectives

## Installing

Using bundler

    $ bundle install

## Running tests

    $ rake spec

## Running application

    $ ruby lib/easy_retro.rb

## Tech Stack

This is the tech stack used for this project

### Languages

* Ruby
* Javascript
* MongoDB
* HTML 5
* CSS 3

### Framework

* Sinatra
* Knockout.js
* JQuery
* Bootstrap
* Jasmine
* Rspec + Capybara
